Installation¶

The following are the steps to install improcpy:

  1. Step 1. improcpy can be imported very easily. First clone the GitHub repo

    git clone git@github.com:hammy4815/improcpy.git

  2. Step 2. Ensure pip is installed on your machine

  3. Step 3. Install locally via pip

    pip install -e .

  4. Step 4. You can now import the different components of improcpy into your python module

    from improcpy import ImageProcessing as ip
from improcpy import GrayImage, ColorImage
